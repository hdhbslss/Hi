隱私權政策（Privacy Policy）

生效日期：2026年6月20日

感謝您使用本機器人（以下稱「本服務」）。本隱私權政策說明本服務如何收集、使用及保護您的資訊。

1. 收集的資訊
為提供服務功能，本服務可能收集：
- Discord 使用者 ID
- Discord 伺服器 ID
- Discord 頻道 ID
- 使用者設定資料
- 指令使用紀錄
- 系統錯誤日誌

2. 資訊用途
收集的資訊可能用於：
- 提供及維護服務功能
- 儲存伺服器設定
- 防止濫用與惡意行為
- 改善服務品質
- 排查系統問題

3. 資料保存
資料將於提供服務所需期間內保存。若不再需要，將於合理時間內刪除。

4. 資料分享
除法律要求外，本服務不會出售、出租或分享您的資料給第三方。

5. 資料安全
我們將採取合理措施保護您的資料，但無法保證網路傳輸或儲存系統的絕對安全。

6. 第三方服務
本服務依賴 Discord 平台運作，因此您同時受到 Discord 相關政策約束。

7. 使用者權利
您可聯繫開發者詢問、修改或刪除與您相關的資料（法律或技術限制除外）。

8. 政策修改
本政策可能因服務更新而調整。修改後版本將於發布時立即生效。

聯絡方式：
請透過官方支援伺服器或開發者提供的聯絡方式與我們聯繫。
